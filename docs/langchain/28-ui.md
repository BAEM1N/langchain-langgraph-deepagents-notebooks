# Agent Chat UI Documentation Summary

## Overview
Agent Chat UI is a Next.js application that "provides a conversational interface for interacting with any LangChain agent." It features real-time chat, tool visualization, time-travel debugging, and state forking capabilities, designed to work with agents created using the `create_agent` function.

## Getting Started

**Hosted Option:**
Users can visit the live version at agentchat.vercel.app, enter their agent's deployment URL or local server address, and begin chatting immediately with automatic tool call rendering.

**Local Development:**
Two setup methods are available:

1. Using npx to create a new project
2. Cloning the repository directly from GitHub

Both approaches require installing dependencies with pnpm and running `pnpm dev` to start development.

## Configuration Steps

To connect Agent Chat UI to an agent, users must provide:
- **Graph ID**: The graph name from the `langgraph.json` file
- **Deployment URL**: The agent server's endpoint (local or deployed)
- **LangSmith API key** (optional): Required only for certain deployment scenarios

## Key Features

- Automatic detection and rendering of tool calls and interrupts
- Built-in support for tool result messages
- Customizable message display options
- Open-source codebase available for adaptation
- Support for generative UI implementation

The interface automatically fetches interrupted threads from connected agents once properly configured.
