# Agent Chat UI Documentation

## Overview

Agent Chat UI is a Next.js application offering a conversational interface for LangChain agents. It features "real-time chat, tool visualization, and advanced features like time-travel debugging and state forking" and integrates seamlessly with agents built using `create_agent`.

## Key Features

- Interactive chat interface for LangChain agents
- Tool call visualization and rendering
- Time-travel debugging capabilities
- State forking functionality
- Support for generative UI
- Open source and customizable

## Getting Started

### Hosted Version
1. Visit the hosted application at agentchat.vercel.app
2. Enter your agent's deployment URL or local server address
3. Begin chatting -- the interface automatically detects and displays tool calls

### Local Development Setup

Two installation options are available:

**Using npx:**
```bash
npx create-agent-chat-app --project-name my-chat-ui
cd my-chat-ui
pnpm install
pnpm dev
```

**Via Git clone:**
```bash
git clone https://github.com/langchain-ai/agent-chat-ui.git
cd agent-chat-ui
pnpm install
pnpm dev
```

## Agent Configuration

To connect Agent Chat UI to your agent, provide:

- **Graph ID**: Your graph name from `langgraph.json`
- **Deployment URL**: Agent server endpoint (e.g., `http://localhost:2024`)
- **LangSmith API key**: Optional for local deployments

The interface automatically retrieves interrupted threads from your configured agent once connected.
